## Инструкция по запуску:
### 1. Запуск сервера:
> java -jar TankGame/target/tank-game.jar
### 2. Запуск клиента:
> java -jar TankGameClient/target/tank-game-client.jar
### Управление:
WASD - move
SPACE - shoot
